# Facturio - Facturation 1-clic

Facturio est une application de facturation pour freelances marocains. Paiements locaux (Wafa Cash, Barid Bank, Stripe), zero erreur, dashboard KPIs.

## Themes

- **Dark** : Marron/Or/Jaune
- **Light** : Beige/Marron/Or

## Développement

```bash
cd frontend
npm install
npm run dev
```

## Déploiement

Pusher sur `main` → GitHub Actions construit et déploie automatiquement sur GitHub Pages.

## Stack

- React 18 + TypeScript
- Vite
- GitHub Pages
